# UniExpress 🍔🍟🍕🌭🍗

## Features😋
### Customer Site
- Login/ Sign up/ Logout
- Update Personal Details
- View Menu
- Order Food
- Receive order confirmation email
- Send contact email

### Admin Panel
- Login/ Sign up/ Logout
- Manage Admin(CRUD)
- Manage Food(CRUD)
- Manage Order(CRUD)

### Frameworks and Libraries used🧐
- Bootstrap
- PHPMailer
- Animate On Scroll
